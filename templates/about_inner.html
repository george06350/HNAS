HNAS 轻量级家庭 NAS 工具
版本：3.0.0
简介：
HNAS 是基于 Python Flask 开发的轻量级 NAS 工具，采用 Windows 风格桌面 UI，多用户隔离，文件管理便捷，支持家庭/个人私有云部署。
功能特色：
  • 🖥️ 桌面/开始菜单风格的 Web UI，支持多窗口、任务栏、窗口最大化/最小化/关闭
  • 👥 多用户注册、登录与独立空间，权限互相隔离（未来计划）
  • 📁 文件管理器支持文件夹浏览、上传、下载
  • 🔒 简单易用，开箱即用，无需数据库
3.0 主要亮点：
  • 全新用户与数据目录结构，安全隔离更规范（用户数据、配置集中于 /data/system/）
  • 新增“system目录”独立浏览入口，超级管理员可直接浏览 /data/system 下全部内容
  • 多账号数据隔离与安全增强
  • 壁纸自定义与预览体验优化
  • 文件管理器与设置界面优化
快速开始：
  1. 环境准备：Python 3.8 及以上，pip
  2. 安装依赖：pip install flask
  3. 运行项目：python app.py
  4. 首次运行自动进入注册页面，默认监听 http://127.0.0.1:5000
